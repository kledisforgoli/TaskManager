## Task Manager

A task manager app that allows users to: 
- Login
- Sign Up
- List all tasks
- Create tasks
- Edit tasks
- Delete tasks
- Filter tasks by date, title, status

### Tech Stack
This project is written in .NET Core, SQL Server
